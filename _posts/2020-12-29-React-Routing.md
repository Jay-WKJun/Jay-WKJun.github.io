# Routing

React는 기본적으로 SPA(Single Page App)가 기본이지만 MPA(Multi Page App)처럼 다른 주소에 다른 page를 보여줄 수 있다.

React는 하나의 HTML에 각 주소에 Routing된 Component들을 통해 새롭게 rendering하여 여러 Page가 존재하는 것 처럼 보여줄 수 있다.
